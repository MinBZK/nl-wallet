Use Case 2.1 Setup a remote PIN and biometric unlock

Overview

Aspect

Description

Summary

The user chooses and confirms a secure 6-digit remote Personal Identification Number (PIN) for the app, and enables biometric unlock if supported and desired.

Goal

Securing the App with PIN and optionally biometrics.

Preconditions

None

Postconditions

  • Remote PIN is configured.
  • If selected, biometric unlock is configured.

Triggered by

  • User completes UC1.1 Introduce the app

Additional Documentation

  • Wallet creation

Possible errors

  • No Internet
  • Server Unreachable
  • Device Insecure

Logical Test Cases

  • LTC61 PIN entries do not match, try again
  • LTC62 PIN entries do not match, choose new PIN
  • LTC63 PIN entry does not conform to policy
  • LTC66 Setup PIN Happy flow
  • LTC67 Setup PIN fails device does not pass app and key attestation
  • LTC74 User skips setting up biometrics
  • LTC75 Device does not support biometrics
