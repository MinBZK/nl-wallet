Partial Flow 2.8 Setup PIN

Overview

Aspect

Description

Summary

The user enters the PIN and confirms it. If entries do not match or if the PIN is too simple the user has to try again.

Goal

Setup a PIN that matches the PIN policy.

Preconditions

None

Postconditions

Depends on the use case.

Used by

  • UC2.1 Setup remote PIN
  • PF2.3.2 Recover PIN
  • PF2.6 Change PIN

Parameters

None

Possible Results

  • PIN setup succeeds
  • Back

Additional Documentation

  • PIN Validation

Possible errors

  • No Internet
  • Server Unreachable
  • Device Insecure
