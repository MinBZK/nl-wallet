You can check which details you already shared. This helps you understand what happened.

**What can you do?**

- Go to activities via overview or menu.
- All activities will be listed
- Tap on one activity to see the details.

**What this means**

- This action is finished.
- You can check what was shared.
- You cannot change what was shared there.
