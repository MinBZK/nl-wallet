You check the website address to avoid a fake website.

**Why is this important?**

- A fake website can trick you into sharing details.
- Checking the address lowers this risk.

**What if the address does not match?**

- Tap "Stop".
- Close the website you opened.
- Go to the organisation website yourself and start again.
- Report it if it looks fake.
