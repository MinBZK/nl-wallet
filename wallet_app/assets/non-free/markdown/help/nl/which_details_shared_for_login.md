Je ziet precies welke gegevens worden gevraagd voordat je inlogt.

**Wat moet je doen?**

- Kijk naar de organisatie.
- Lees welke gegevens worden gevraagd.
- Tik op "Bekijk details" om meer te zien.
- Ga alleen verder als dit duidelijk is.
