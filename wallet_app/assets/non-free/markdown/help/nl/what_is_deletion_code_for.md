Je gebruikt deze code om je wallet te verwijderen als je je telefoon kwijt bent of als die is gestolen.

**Hoe werkt het?**

- Je gebruikt de code op de NL Wallet-website.
- Daarna werkt de wallet op deze telefoon niet meer.
- Je kunt deze code niet gebruiken om de app te openen.
