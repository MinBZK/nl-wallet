De verwijder-code werkt alleen om je wallet te verwijderen.

**Wat moet je doen?**

- Open de NL Wallet-website in je browser.
- Kijk of je de code goed hebt opgeschreven.
- Vul de code in op de NL Wallet-website.
- Kijk de code nog een keer na als het nog niet werkt.
