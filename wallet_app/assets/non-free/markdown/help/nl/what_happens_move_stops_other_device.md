Op je nieuwe telefoon komt geen wallet te staan. Je wallet blijft op je oude telefoon.

**Wat moet je weten?**

- Je nieuwe telefoon krijgt je wallet niet.
- Je oude telefoon houdt je wallet.
- Je kunt later opnieuw beginnen.
