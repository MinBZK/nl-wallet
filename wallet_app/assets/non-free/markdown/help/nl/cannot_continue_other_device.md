Het verplaatsen wacht nog op bevestiging op de telefoon waar je wallet nu op staat.

**Wat moet je doen?**

- Kijk op welke telefoon je wallet nu staat.
- Pak die telefoon.
- Kies "Ja, verplaatsen".
- Vul je pincode in.
- Houd beide telefoons dicht bij elkaar.
- Wacht op de volgende stap.
