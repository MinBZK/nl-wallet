Je wallet is nog niet klaar voor gebruik als je pincode nog niet is ingesteld.

**Wat moet je weten?**

- Er wordt niets bewaard tot je klaar bent met het instellen van je pincode.
- Je kunt deze wallet pas gebruiken als je pincode klaar is.
- Je hebt later ook nog een pincode nodig als je gezicht-scan of vingerafdruk gebruikt.
