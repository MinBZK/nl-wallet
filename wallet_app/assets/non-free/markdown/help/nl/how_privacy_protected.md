Jij houdt de controle. Jij opent zelf je wallet. Jij kijkt wat je deelt. Jij geeft akkoord met je pincode.

**Goed om te weten**

- Je kunt altijd stoppen.
- Je opent NL Wallet met je eigen pincode.
- Je kijkt elke keer wat je deelt.
- Je geeft akkoord met je pincode voordat je deelt.
- Je ziet altijd wat er wordt gevraagd voordat je akkoord geeft.
