De demo is alleen een voorbeeld. Niets in de demo is echt.

**Waarvoor is dit?**

- De demo laat zien hoe NL Wallet werkt.
- Je kunt de stappen proberen.

**Wat is niet echt?**

- De demo gebruikt testgegevens.
- De demo gebruikt je echte kaarten niet.
- De demo gebruikt je echte gegevens niet.

**Wat kun je in de demo doen?**

- Je kunt door de stappen gaan.
- Je kunt de knoppen proberen.
- Je kunt de demo sluiten als je klaar bent.

**Wat kun je niet in de demo doen?**

- Je kunt niet inloggen bij echte organisaties.
- Je kunt geen echte gegevens delen.

**Wordt opgeslagen wat je doet?**

- Tijdens een test kan de app opslaan wat je in de demo doet.
- Dit helpt om de app beter te maken.
