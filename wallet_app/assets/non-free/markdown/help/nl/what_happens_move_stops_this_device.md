Het verplaatsen stopt. Je wallet blijft op je oude telefoon staan.

**Wat moet je weten?**

- Je oude telefoon heeft je wallet nog.
- Je wallet gaat niet naar je nieuwe telefoon.
- Je kunt NL Wallet op je oude telefoon blijven gebruiken.
- Je kunt later opnieuw beginnen.
