Schrijf de verwijder-code op papier en bewaar die op een veilige plek.

**Wat moet je doen?**

- Schrijf de code op papier.
- Bewaar het papier op een veilige plek.
- Houd de code privé.
- Deel de code niet.
- Tik op "Ik heb het opgeschreven" om verder te gaan.
