Meldingen op je telefoon helpen je om belangrijke dingen op tijd te zien.

**Wat moet je weten?**

- Je ziet belangrijke updates sneller.
- Tik je op een melding? Dan opent de juiste pagina.
- Tik je op een banner? Dan opent ook de juiste pagina.
- Zo kun je op tijd iets doen.
