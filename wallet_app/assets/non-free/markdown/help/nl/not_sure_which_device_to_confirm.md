Bevestig het verplaatsen op de telefoon waar je wallet nu op staat.

**Wat moet je doen?**

- Pak je andere telefoon.
- Op die telefoon staat je wallet nu nog.
- Kies "Ja, verplaatsen".
- Vul je pincode in.
- Houd beide telefoons dicht bij elkaar.
