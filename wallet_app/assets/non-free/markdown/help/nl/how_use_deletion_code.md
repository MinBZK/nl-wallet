Je kunt je verwijder-code op 2 manieren gebruiken.

**Op de NL Wallet-website**

- Ga naar de NL Wallet-website.
- Vul je verwijder-code van 18 tekens in.
- Doorloop de controlestap.
- Wacht tot je ziet dat het is gelukt.

**Aan de telefoon**

- Houd je verwijder-code klaar.
- Bel de helpdesk.
- Vraag de helpdesk om je NL Wallet te verwijderen.
- Zeg je verwijder-code van 18 tekens.

**Wat gebeurt er daarna?**

- Je NL Wallet wordt verwijderd zodra je telefoon internet heeft.
