Je ziet precies welke gegevens worden gevraagd voordat je verdergaat.

**Wat moet je doen?**

- Lees de delen per kaart onder de gevraagde gegevens.
- Tik op "Bekijk details" om de hele lijst van die kaart te zien.
- Stop als je deze gegevens niet wilt delen.
