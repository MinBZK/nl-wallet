Als de geblokkeerde wallet wordt verwijderd, verdwijnen de walletgegevens ook van deze telefoon.

**Wat moet je weten?**

- Dit zijn ook je kaarten.
- Dit zijn ook je kaartgegevens.
- Dit zijn ook je activiteiten.
- Zo blijft je wallet op deze telefoon beschermd.
