Je wallet blijft op je andere telefoon staan.

**Wat moet je doen?**

- Tik op "Sluiten".
- Blijf je oude telefoon gebruiken.
- Je nieuwe telefoon heeft je wallet niet gekregen.
- Start het verplaatsen opnieuw als je je wallet nog hierheen wilt verplaatsen.
