Op de telefoon met je huidige NL Wallet staat je wallet nu.

**Hoe weet je dat?**

- Dat is je oude telefoon.
- Op die telefoon zie je de stap om te bevestigen.
- Op die telefoon gebruik je je pincode om akkoord te geven.
