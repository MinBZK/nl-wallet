Je kunt je verwijder-code alleen zien als je je wallet nog kunt openen.

**Wat moet je doen?**

- Open Menu.
- Open Instellingen en beveiliging.
- Tik op "Bekijk je verwijdercode".
- Schrijf de code op papier.
- Bewaar die op een veilige plek.
